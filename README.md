# Курсовой проект от Нетологии «HTML-вёрстка: с нуля до первого макета»

Проект был сделан в песочнице Codepen.io: [https://codepen.io/Yulia-Aseeva/pen/xxmYWbd](https://codepen.io/Yulia-Aseeva/pen/xxmYWbd).

Рада представить мой курсовой проект по HTML верстке сайта с нуля. Мой проект представляет собой полноценный блог, ориентированный на творчество, спорт и образ жизни. Я создала его с нуля, используя HTML и CSS, в рамках курсового проекта от Нетологии. Сайт имеет стильный и современный дизайн, а также ряд интересных функций. Например, при наведении на заголовки они меняют цвет, что делает интерактивное взаимодействие с контентом более приятным для пользователя. Кроме того, при наведении на текст появляется нижнее подчеркивание, что добавляет дополнительный визуальный акцент. Этот проект не только демонстрирует мои навыки в HTML и CSS, но и позволяет мне проявить креативность и внести свой вклад в создание интересного и уникального онлайн-сообщества.
